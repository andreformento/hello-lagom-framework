# hello-lagom-framework

Simple example with Lagom Framework

See implementation with

- [Scala](whith-scala)
- [Java](whith-java)

## Requirements

- Java 8
- sbt 1.x

## Show me the way

Make your choice: `cd with-scala` or `cd with-java` and then

```
sbt
... (booting up)
> runAll
```

To see your new child, go to: http://localhost:9000/api/hello/World
