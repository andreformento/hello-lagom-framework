# hello-lagom-framework

Simple example with Lagom Framework (Scala)

## Requirements

- Java 8
- sbt 1.x

## Show me the way

```
cd hello-lagom-framework
sbt
... (booting up)
> runAll
```

To see your new child, go to: http://localhost:9000/api/hello/World
